Yawords

Download this project as a .zip file Download this project as a tar.gz file

Букмарклет для повторения слов из Тетрадок Яндекс.Словарей. Заменяет стандартный набор слов блока "Проверь себя" словами из Ваших тетрадок.

Установка

Перетащите ссылку yawords в панель закладок Вашего браузера.

Если Вы используете Microsoft Internet Explorer, то кликните на ссылке правой кнопкой мышки и выберите пункт "Добавить в папку Избранное". После этого найдите закладку yawords в "Избранном", снова кликните на ней правой кнопкой мышки и выберите пункт "Добавить на панель избранного".

Использование

  1. Зайдите на страницу Яндекс.Словарей
  2. Найдите на панели закладок своего браузера закладку yawords и кликните на нее.

Известные проблемы

  1. Нельзя прослушать многосложные слова (к примеру, "set out", "help on");
  2. Если тетрадки не загружаются, попробуйте разлогиниться и снова залогиниться на сайте Яндекса. В случае обнаружения каких-либо проблем с использованием скрипта, просьба сообщать о них в баг-трекер.
